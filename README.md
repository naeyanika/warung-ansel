# Warung Ansel

Ini adalah proyek iseng yang aku buat untuk form input siapa saja yang mau nitip sarapan.  
Frontend dibuat dengan **React**, backend menggunakan **Supabase**.

## Fitur
- Form input untuk nitip sarapan
- Integrasi dengan Supabase untuk backend

## Teknologi
- **Frontend**: React
- **Backend**: Supabase

## Cara Menjalankan

1. **Frontend**  
   - Pastikan kamu sudah install Node.js dan npm/yarn  
   - Jalankan di folder frontend:  
     ```bash
     npm install
     npm start
     ```

2. **Backend (Supabase)**  
   - Buat project di [Supabase](https://supabase.com/)  
   - Konfigurasikan koneksi Supabase di frontend

## Catatan
Proyek ini hanya dibuat iseng untuk belajar dan memudahkan koordinasi titip sarapan :)
